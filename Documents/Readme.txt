ABOUT
	This is a syntactic parser for Tamil. It is used to identify the syntactic constituents of a sentence as a tree structure.

REQUIREMENTS
	* J2RE 1.3 or higher
	* Operating System: Windows/Linux
	* Any TAB Font should be installed in the System to manipulate Tamil text

Main class : ParserUI.java
